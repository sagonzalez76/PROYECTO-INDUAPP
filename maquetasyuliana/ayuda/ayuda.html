Para ingresar a Apprendiz, cada usuario debera llenar todos los formularios de incripcion y de esta manera podra comenzar el uso del mismo.

Para recuperar contraseña, el usuario debera ingresar su correo electronico, en la pagina de registro para si enviarle un correo de recuperacion de contraseña.

Si quieres ingresar a APPRENDIZ, escribe tus datos y correo electronico, en este recibiras una contraseña y nombre de usuario para ingresar a esta plataforma.
